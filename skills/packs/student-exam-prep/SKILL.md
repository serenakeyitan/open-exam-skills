# Student Exam Prep Pack

Use this workflow when a student wants a full study flow from notes.

## Workflow

1. Create a **mindmap** from notes.
2. Convert the mind map into **flashcards**.
3. Generate a **quiz** from the same material.
4. Highlight weak areas and suggest a review loop.

## Output Contract

- Provide outputs in the order above.
- Keep prompts short and exam-focused.
